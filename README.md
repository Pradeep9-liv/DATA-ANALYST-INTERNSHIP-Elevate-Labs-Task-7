# Sales Performance Dashboard (Power BI)

## Project Overview
This project demonstrates the creation of an interactive Business Intelligence dashboard using **Power BI Desktop**.  
The dashboard analyzes sales performance, profitability, and trends using a structured sales dataset.

---

## Dataset
- Source: Kaggle (Sales dataset)
- Format: CSV
- Key fields include:
  - Sale_Date
  - Region
  - Product_Category
  - Sales_Amount
  - Quantity_Sold
  - Unit_Cost
  - Customer_Type
  - Sales_Channel

---

## Tools Used
- Power BI Desktop
- Microsoft Excel (for initial data inspection)

---

## Workflow
1. Imported CSV dataset into Power BI and validated data types.
2. Created calculated measures using DAX:
   - Total Sales
   - Total Quantity Sold
   - Total Cost
   - Total Profit
   - Profit Margin
3. Built core visuals:
   - Sales by Product Category
   - Sales by Region
   - Profit Trend Over Time
4. Added interactive slicers for Region, Customer Type, Sales Channel, and Year.
5. Designed KPI cards for Total Sales, Total Profit, and Total Quantity.
6. Applied professional formatting and validated dashboard interactivity.
7. Exported final dashboard as PBIX and PDF.

---

## Key Insights
- Identified top-performing product categories by total sales.
- Compared regional sales contributions.
- Analyzed profit trends over time to understand business performance.
